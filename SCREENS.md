# RPO-SaaS MVP 画面一覧

## 認証
1. **ログイン画面** (`/login`)
   - Email、パスワード入力
   - JWT保存

2. **ユーザー登録画面** (`/register`)
   - テナント選択（既存）またはテナント新規作成
   - Email、名前、パスワード入力

## ダッシュボード
3. **ホーム/ダッシュボード** (`/dashboard`)
   - サマリー指標表示（求人数、掲載中件数、応募数など）
   - 最近のアクティビティ
   - 承認待ち件数バッジ

## 顧客管理
4. **顧客一覧画面** (`/dashboard/customers`)
   - テーブル表示（名前、求人数、作成日）
   - 検索・フィルタ
   - 「新規作成」ボタン

5. **顧客作成・編集画面** (`/dashboard/customers/new`, `/dashboard/customers/:id/edit`)
   - フォーム（名前、説明）
   - バリデーション

## 求人管理
6. **求人一覧画面** (`/dashboard/jobs`)
   - テーブル表示（タイトル、顧客、ステータス、更新日）
   - ステータスフィルタ（全て、下書き、承認待ち、掲載中など）
   - 「新規作成」ボタン

7. **求人作成・編集画面** (`/dashboard/jobs/new`, `/dashboard/jobs/:id/edit`)
   - 基本情報フォーム
     - タイトル（必須）
     - 顧客選択（必須）
     - 職種、勤務地、給与、雇用形態、応募要件
   - バリデーション

8. **求人詳細画面** (`/dashboard/jobs/:id`)
   - 基本情報表示
   - ステータス表示
   - 「テキスト生成」「画像生成」ボタン
   - 「承認申請」ボタン
   - タブ切り替え
     - 生成履歴
     - 掲載状況
     - 問い合わせ

## テキスト・画像生成
9. **テキスト生成画面** (`/dashboard/jobs/:id/generate-text`)
   - プロンプト入力（任意）
   - 「生成実行」ボタン
   - 生成結果プレビュー
   - 「採用」ボタン

10. **画像生成画面** (`/dashboard/jobs/:id/generate-image`)
    - プロンプト入力（任意）
    - 「生成実行」ボタン
    - 生成結果プレビュー
    - 「採用」ボタン

11. **生成履歴・差分確認画面** (`/dashboard/jobs/:id/history`)
    - バージョン一覧
    - 差分表示（テキストの場合）
    - サムネイル表示（画像の場合）

## 承認フロー
12. **承認待ち一覧画面** (`/dashboard/approvals`)
    - テーブル表示（求人タイトル、申請日、申請者）
    - 「詳細」ボタン

13. **承認詳細画面** (`/dashboard/approvals/:id`)
    - 求人情報表示
    - テキスト・画像プレビュー
    - 「承認」「差戻し」ボタン
    - コメント入力欄

## 媒体掲載
14. **掲載状況一覧画面** (`/dashboard/publications`)
    - テーブル表示（求人、媒体、ステータス、掲載日）
    - 「掲載実行」ボタン（承認済み求人選択）

15. **掲載詳細・ログ画面** (`/dashboard/publications/:id`)
    - 掲載情報
    - ステータス
    - ログ一覧（アクション、日時、結果、エラーメッセージ）
    - 「更新」「停止」「リトライ」ボタン

## 問い合わせ対応
16. **問い合わせ一覧画面** (`/dashboard/inquiries`)
    - テーブル表示（求人、応募者、カテゴリ、ステータス、受信日）
    - ステータスフィルタ

17. **問い合わせ詳細画面** (`/dashboard/inquiries/:id`)
    - 問い合わせ内容表示
    - 自動分類結果
    - 返信案一覧
    - 「返信案生成」ボタン
    - 「送信」ボタン

18. **返信案編集・送信画面** (`/dashboard/inquiries/:id/response`)
    - 返信案テキストエリア（編集可能）
    - 「送信」ボタン
    - 送信履歴

## 日程調整
19. **日程調整一覧画面** (`/dashboard/schedules`)
    - テーブル表示（候補者、ステータス、作成日）
    - 「新規作成」ボタン

20. **日程調整作成画面** (`/dashboard/schedules/new`)
    - 候補者情報入力（名前、Email）
    - 問い合わせ選択（任意）
    - 「候補日時生成」ボタン（ルールベースで3つ提示）
    - 候補日時プレビュー
    - 「送信」ボタン

21. **日程調整確定画面（候補者向け）** (`/schedules/:id/select`)
    - 候補日時3つ表示
    - 選択ラジオボタン
    - 「確定」ボタン

22. **日程調整詳細画面** (`/dashboard/schedules/:id`)
    - 候補日時一覧
    - 確定日時
    - カレンダーイベントID
    - ステータス

## 分析ダッシュボード
23. **分析画面** (`/dashboard/analytics`)
    - 日付範囲選択
    - フィルタ（求人、媒体）
    - 指標カード
      - 表示数合計
      - クリック数合計
      - 応募数合計
      - クリック率平均
      - 応募率平均
    - グラフ（時系列）
      - 表示数・クリック数・応募数の推移
    - テーブル（求人×媒体別の詳細）

## 設定・その他
24. **ユーザー設定画面** (`/dashboard/settings`)
    - プロフィール編集
    - パスワード変更

25. **監査ログ画面** (`/dashboard/audit-logs`)
    - テーブル表示（日時、ユーザー、アクション、リソース）
    - 検索・フィルタ

---

## 画面遷移フロー（主要導線）

```
ログイン
  ↓
ホーム/ダッシュボード
  ↓
顧客一覧 → 顧客作成
  ↓
求人一覧 → 求人作成
  ↓
求人詳細
  ├→ テキスト生成 → 生成履歴・差分確認
  ├→ 画像生成 → 生成履歴・差分確認
  └→ 承認申請
       ↓
承認待ち一覧 → 承認詳細 → 承認/差戻し
  ↓（承認後）
掲載状況一覧 → 掲載実行 → 掲載詳細・ログ
```

## UI/UXの仮置き方針
- デザインシステム: Tailwind CSS + shadcn/ui
- テーブル: TanStack Table（ソート、ページネーション）
- フォーム: React Hook Form + Zod（バリデーション）
- 通知: Toast（成功・エラー）
- ローディング: Spinner、Skeleton
- レスポンシブ対応（最低限）
